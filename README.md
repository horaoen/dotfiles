# dotfiles

# Get Started
## dependencies

```Bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew insall fd
brew install cloc
brew install git
brew install tmux
brew install jenv
brew install nvm
brew install lazygit
```

## start
```
chmod 755 ./install.sh
./install.sh
```
